<h3 align="center">
A Node.js + TypeScript + Express + Socket.io API for **simulated real-time stock trading**. 
View live stock prices, buy/sell stocks, and track your portfolio. Perfect for beginners to learn REST APIs and WebSockets.
</h3>
<br/>

## 🚀 Features

- **REST endpoints** for stocks, portfolio, buy & sell operations  
- **WebSocket server** for real-time stock updates  
- **In-memory database** for temporary storage  
- Fully commented **TypeScript code**  
- **Simulated stock price changes** every second
